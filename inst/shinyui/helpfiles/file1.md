# Loading Data

Select you data in csv format for upload. The data must have the following 
information in it:

1. Subject ID: A unique identifier for each subject.
2. Either the ID of the subject's recruiter or RDS coupon information (i.e. 
their coupon and the coupons they were given)
3. Network Size: The RDS network size variable (aka degree), which is the 
of people in the population the subject knows and could potentially have passed 
a coupon to.
4. Subject's Hashed Identifier: The privatized identifier for the subject.
5. Hashed Identifiers For Contacts: The privatized identifiers for each 
contact identified by the subject.
